# Pokedex TCG

## 프로젝트 소개

Pokedex TCG는 포켓몬 카드 정보를 실시간 검색하고 3D 인터랙션으로 카드 UI를 탐색할 수 있는 웹 플랫폼입니다.  
사용자가 포켓몬 카드를 검색하고 다양한 카드 정보를 확인할 수 있으며, 카드 UI에 3D 인터랙션을 적용하여 보다 직관적이고 몰입감 있는 사용자 경험을 제공하는 것을 목표로 개발되었습니다.

# 담당 역할 (Frontend)

## 서버 컴포넌트를 활용한 초기 로딩 성능 개선

클라이언트 렌더링으로 인해 발생하는 초기 로딩 성능 문제를 해결하기 위해 Next.js Server Component를 도입했습니다.  
HTML을 서버에서 미리 렌더링하도록 구조를 변경하여 초기 로딩 속도를 개선했습니다.

기존 컴포넌트 구조를 서버 컴포넌트와 클라이언트 컴포넌트로 분리하고 Lighthouse 성능 테스트를 통해 검증한 결과,  
FCP(First Contentful Paint)를 1.9초에서 300ms로 단축하여 약 84% 성능 개선을 달성했습니다.

---

## CSS 3D Transform을 활용한 카드 인터랙션 구현

정적인 카드 UI의 한계를 개선하기 위해 마우스 드래그 제스처 기반의 3D 카드 인터랙션을 구현했습니다.

CSS transform3d와 perspective 속성을 활용하여 GPU 가속 기반의 렌더링을 적용했고  
60fps의 부드러운 애니메이션을 유지하도록 구현했습니다.

또한 카드 회전 각도를 -20° ~ 20° 범위로 제한하고  
cubic-bezier easing을 적용하여 자연스러운 인터랙션을 제공했습니다.

---

## 이미지 Priority 최적화를 통한 LCP 개선

Lighthouse 성능 점수 60점과 LCP 지연 문제를 해결하기 위해 Next.js Image 컴포넌트의 priority 속성을 활용했습니다.

초기 화면에서 가장 먼저 노출되는 카드 이미지에 우선 로딩을 적용하여  
불필요한 이미지 로딩을 줄이고 렌더링 우선순위를 조정했습니다.

이를 통해 LCP를 2초 이상에서 0.4초로 개선하고  
Lighthouse 성능 점수 95점을 달성했습니다.
