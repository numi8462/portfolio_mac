# 🐾 Meowhere (어디가냥)

## 프로젝트 소개

Meowhere는 다양한 체험 활동을 탐색하고 예약할 수 있는 웹 플랫폼입니다.  
사용자는 지도 기반 탐색을 통해 주변 체험 활동을 확인하고 원하는 날짜에 예약할 수 있으며, 후기 작성과 공유 기능을 통해 체험 경험을 기록하고 다른 사용자와 공유할 수 있습니다.

## 프로젝트 기간

2025.05.27 ~ 2025.06.26

---

# 담당 역할 (Frontend)

## 공통 UI 컴포넌트 구현

프로젝트 전반에서 재사용 가능한 공통 Modal 컴포넌트를 구현했습니다.  
전역 상태를 활용하여 어느 페이지에서도 호출할 수 있도록 설계하여 로그인, 회원가입 등 다양한 기능에서 재사용할 수 있는 구조로 구현했습니다.

## 로그인 / 회원가입 통합 모달

로그인과 회원가입 기능을 하나의 모달에서 처리할 수 있도록 UI를 설계했습니다.  
상태 관리 로직을 통해 로그인과 회원가입 화면을 자연스럽게 전환할 수 있도록 구현했습니다.

## 로그인 상태 관리

사용자 인증 상태를 전역 상태로 관리하여 여러 페이지에서 로그인 상태를 일관되게 유지하도록 구현했습니다.  
로그인 여부에 따라 UI가 조건적으로 렌더링되도록 처리했습니다.

## Next.js API Route 구현

Next.js의 API Route를 활용하여 로그인 관련 API 처리 및 외부 API와의 중간 처리 로직을 구현했습니다.  
이를 통해 클라이언트와 서버 로직을 분리하여 구조적으로 관리할 수 있도록 구성했습니다.

## Toast 알림 시스템 구현

사용자의 주요 행동에 대한 피드백을 제공하기 위해 Toast 알림 시스템을 구현했습니다.  
로그인, 찜 기능 등 주요 액션이 발생했을 때 사용자에게 상태를 전달할 수 있도록 공통 Toast 컴포넌트를 구성했습니다.

## 찜(좋아요) 기능 및 찜 목록 페이지 구현

사용자가 관심 있는 체험을 저장할 수 있도록 찜 기능을 구현했습니다.  
또한 사용자가 저장한 체험 목록을 확인할 수 있는 찜 목록 페이지를 개발했습니다.

## 카카오 OAuth 로그인 연동

카카오 OAuth를 활용한 소셜 로그인 기능을 구현하여 사용자가 간편하게 로그인할 수 있도록 했습니다.

## 카카오 공유 기능 구현

카카오 공유 API를 활용하여 체험 정보를 다른 사용자에게 쉽게 공유할 수 있는 기능을 구현했습니다.
