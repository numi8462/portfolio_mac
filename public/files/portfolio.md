# Mac-포트폴리오

## 프로젝트 개요

맥OS 스타일의 포트폴리오 웹사이트입니다. Finder, Dock, Navbar 등 macOS 인터페이스를 웹으로 구현하여 독특한 사용자 경험을 제공합니다.

## 주요 기능

- **Finder**: 프로젝트 파일을 탐색하는 듯한 인터페이스
- **Dock**: 맥 스타일의 Dock 앱 실행바
- **Window System**: 드래그 가능한 창 시스템
- **반응형 디자인**: 데스크탑 최적화

## 사용 기술

- React 19
- TypeScript
- Tailwind CSS
- GSAP (애니메이션)
- Zustand (상태 관리)
- Vite
