# BPMN Web Application (DeHeus)

## 프로젝트 소개

학교 졸업 프로젝트로 여러 해외 기업에서 의뢰한 프로젝트 중 하나를 선택하여 팀 단위로 개발을 진행했습니다.  
총 5개의 프로젝트를 선택하여 각 기업에 포트폴리오를 제출했고, 그중 DeHeus 회사로부터 연락을 받아 인터뷰를 진행하게 되었습니다.

인터뷰 이후 프로젝트의 세부 사항과 Functional Requirements(기능 요구사항)를 전달받았으며, 프로젝트가 최종 확정된 후 회사와 함께 필수 기능과 개발 방향에 대해 논의했습니다. 개발 기간 동안 매주 오프라인과 온라인 미팅을 병행하며 프로젝트 진행 상황을 공유했습니다.

개발 방식은 Agile 방법론을 기반으로 진행되었으며, 저는 프론트엔드 개발을 담당했습니다.

---

# 개발 배경

DeHeus 회사는 비즈니스 프로세스 모델(Business Process Model)을 자주 사용하는 기업입니다.  
기존에는 Excel과 Bizagi 도구를 사용하여 BPMN 다이어그램을 작성했지만 다음과 같은 문제점이 있었습니다.

- BPMN 다이어그램을 생성하는 과정이 복잡하고 반복적인 설정이 필요함
- 자주 사용하는 용어 및 기능을 매번 다시 설정해야 하는 불편함
- Parent - Child 구조의 다이어그램 트리 구조 필요
- 상위 다이어그램과 하위 다이어그램을 명확하게 구분할 수 있는 기능 필요
- 문서 공유 및 공동 작업을 위한 온라인 다이어그램 업데이트 기능 필요
- 웹 환경에서 다이어그램 생성 및 수정 기능 필요

이러한 문제를 해결하기 위해 DeHeus 내부에서 사용할 수 있는 **BPMN 웹 기반 다이어그램 관리 시스템**을 개발하게 되었습니다.

---

# 주요 기능

- BPMN 다이어그램 생성 및 수정 기능
- Parent - Child 구조의 다이어그램 트리 관리
- 상위 / 하위 다이어그램 관계 시각화
- 웹 기반 다이어그램 편집 기능
- 다이어그램 공유 및 협업 기능
- 온라인 환경에서 실시간 다이어그램 업데이트

---

# 담당 역할 (Frontend)

- React 기반 BPMN 다이어그램 UI 구현
- BPMN.js 라이브러리를 활용한 다이어그램 렌더링 및 편집 기능 구현
- 다이어그램 트리 구조 UI 개발
- 사용자 인터랙션 기반 다이어그램 편집 기능 구현
- 백엔드 API와 연동하여 다이어그램 데이터 관리
