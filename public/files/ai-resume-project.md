# AI Resume Analyzer

AI Resume Analyzer is a smart tool that helps you perfect your resume with instant feedback.

Instead of guessing what recruiters want, you get AI-powered insights on keywords, formatting, and overall impact.

Think of it like having a career coach—pointing out strengths, fixing weaknesses, and boosting your chances of landing interviews.

## Features

- **AI-Powered Analysis**: Get instant feedback on your resume
- **Keyword Optimization**: Improve your chances of passing ATS
- **Format Suggestions**: Professional formatting recommendations
- **Impact Scoring**: Measure the effectiveness of your content

## Tech Stack

- Next.js
- Tailwind CSS
- AI Integration

## Getting Started

```bash
npm install
npm run dev
```
